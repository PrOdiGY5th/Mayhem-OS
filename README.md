# Mayhem-OS
Mayhem OS is a Python-powered terminal operating environment that launches and manages multiple applications through command-line controls. It features a custom shell, dynamic app discovery, SnakeFileManager integration, redundancy systems, recovery layers, and built-in file management utilities for a CLI desktop experience.
THE ABOVE DESCRIPTION IS A BIT MISLEADING THE WHOLE PROJECT IS JUST A BUNCH OF APPS THAT CAN BE USED FROM THE main.py FILE WHICH IS A TERMINAL LIKE INTERFACE ONLY FOR USING THE APPS. AND NO IT DOES NOT HAVE A CUSTOM LANGUAGE LIKE BASH SHELL.
